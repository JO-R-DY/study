### [git](https://git-scm.com/)
### [github](https://github.com/)


#### 초기세팅
```
git init
```
- 맥
```
git config --global core.autocrlf input
```
- 윈도우
```
git config --global core.autocrlf true
```
```
git config --global user.name '별명'
```
```
git config --global user.email '이메일주소'
```

- 등록정보 확인하기
```
git --global --list
```

- 버전관리상태확인하기
```
git status
```

- 버전추가하기
```
git add .
```

- 버전생성
```
git commit -m '버전이름'
```

- 버전히스토리 확인
```
git log
```
