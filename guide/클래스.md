## 1. header
1. logo : h1>a>img
2. gnb : ul(depth1)>li(th1)>a,ul(depth2)>li(th2)>a
3. icon_box

```html
<header>
    <div class="logo"></div>
    <div class="gnb">
      <ul class="depth1">
        <li class="th1">
          <a href="javascript:void(0)"></a>
          <ul class="depth2">
            <li class="th2">
              <a href="javascript:void(0)"></a>
            </li>
          </ul>
        </li>
        <li class="th1">
          <a href="javascript:void(0)"></a>
          <ul class="depth2">
            <li class="th2">
              <a href="javascript:void(0)"></a>
            </li>
          </ul>
        </li>
      </ul>
    </div>
    <div class="icon_box"></div>
  </header>
```
## 2. main

```
<main>
  <section class="이름">
  <section class="이름">
  <section class="이름">
</main>
```
